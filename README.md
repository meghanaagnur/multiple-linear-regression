##  What is Multiple Linear Regression?

Multiple Linear Regression is a supervised learning algorithm used to predict a continuous value using **two or more independent variables**.

The model learns the relationship between multiple input features and a target variable by fitting the equation:

**y = b₀ + b₁x₁ + b₂x₂ + ... + bₙxₙ**
